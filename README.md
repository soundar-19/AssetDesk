## üéØ Overview

AssetDesk is a modern, full-stack enterprise asset management system designed to streamline IT asset lifecycle management, from procurement to retirement. Built with Spring Boot and Angular, it provides a comprehensive solution for organizations to track, manage, and optimize their IT resources.

### Key Capabilities
- üìä **Asset Lifecycle Management** - Complete tracking from procurement to disposal
- üé´ **Request Management** - Streamlined asset request and approval workflows
- üîß **Service Records** - Maintenance and service history tracking
- üìà **Analytics & Reporting** - Real-time insights and comprehensive reports
- üîî **Smart Notifications** - Automated alerts for warranty expiry and maintenance
- üë• **Multi-role Support** - Role-based access control for different user types

## üì¶ Installation & Setup

### Prerequisites
- ‚òï Java 21 or higher
- üêò PostgreSQL 12 or higher
- üì¶ Node.js 18 or higher
- üÖ∞Ô∏è Angular CLI 20.x

### Backend Setup

1. **Clone the repository**
```bash
git clone <repository-url>
cd AssetDesk/backend
```

2. **Configure Database**
```bash
# Create PostgreSQL database
createdb assetdesk

# Update application.properties with your database credentials
```

3. **Build and Run**
```bash
# Build the application
./mvnw clean install

# Run the application
./mvnw spring-boot:run
```

The backend will start on `http://localhost:8080`

### Frontend Setup

1. **Navigate to frontend directory**
```bash
cd AssetDesk/frontend
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
ng serve
```

The frontend will start on `http://localhost:4200`

### Docker Setup (Optional)

```bash
# Build and run with Docker Compose
docker-compose up -d
```
